# **AnonymousAuthenticationFilter(익명사용자 인증 필터)**

- 익명사용자 인증 처리 필터이다.
- 익명사용자와 인증한 사용자를 구분한 후 처리하기 위해서 사용한다.
- 화면에서 인증 여부를 구현할 때 

사용자가 인증을 받게되면 세션에 인증을 받은 사용자에 user객체를 저장한 다음 사용자가 자원에 접근을 하려고 하면 세션에서 사용자가 인증된
